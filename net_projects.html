Virtual furnishing
platform

I did this project for Bevir Oy (and the platform is called: BevirPortalApp). Bevir needed platform where they could interact with the orders that their customers create and one place to maintain them.

In the application:

Users are able to:
  • Sign up to the application
  • Update and delete their profile information
Admins are able to:
  • Maintain all orders
  • Download customer images in order to edit them
  • Upload edited images for customer approve
  • Stay up to date every order statuses
Customers are able to:
  • Create new orders
  • Track their order
  • Download ready images
  • Approve order
  • Ask for more edits if unhappy
  • Get discounts on first order and after x amount of orders
  • Ready orders stay in the application customers are always able to come back
This project is done using .Net6 (c#), Entity Framework, MVC, ASP Pages, MS-SQL and is hosted in Azure. I created the whole backend for the application. Unfortunately this projects code is not public.
